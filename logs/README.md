# Logs des sessions Lean

Ce dossier contiendra les traces (logs) de votre travail au cours des séances sur ces sujets de TP. Une fois le module terminé, il vous sera demandé de les mettre à disposition de l'enseignant par un moyen qui sera précisé ultérieurement. 

Cette fonctionnalité est expérimentale : si le dossier occupe trop d'espace disque sur votre compte, il faudra le supprimer une fois les fichiers envoyés.
